# 🌟 About SoonComming

SoonComming is a full-featured e-commerce platform built with Laravel and PHP, designed to help businesses launch, manage, and scale online stores with ease.

The platform provides everything needed to operate a professional online marketplace, including product management, order tracking, customer accounts, payment processing, inventory monitoring, and administrative controls.

Whether you're launching a startup store or managing a large online business, SoonComming provides the tools required to deliver a seamless shopping experience.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/sooncomming.git
cd sooncomming
```

## Install Dependencies

```bash
composer install
npm install
npm run build
```

## Environment Setup

```bash
cp .env.example .env
php artisan key:generate
```

Configure your database:

```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=sooncomming
DB_USERNAME=root
DB_PASSWORD=
```

---

## Database Migration

```bash
php artisan migrate --seed
```

---

## Run Application

```bash
php artisan serve
```

Visit:

```text
http://localhost:8000
```
